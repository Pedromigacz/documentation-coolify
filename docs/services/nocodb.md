# What is NocoDB?

<ZoomableImage src="/docs/images/services/nocodb1.webp" alt="Nocodb dashboard" />

NocoDB is an open source Airtable alternative. Turns any MySQL, PostgreSQL, SQL Server, SQLite & MariaDB into a smart-spreadsheet.
